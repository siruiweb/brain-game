# 🧠 头脑特工队

**Project: VALHALLA** | 人生365天模拟游戏

## 🎮 游戏介绍

一款模拟人生的大脑情绪管理游戏，玩家需要平衡七情六欲，经历5个章节，最终完成365天的人生挑战。

## ✨ 游戏特色

- 🧬 **7种天赋**选择
- 📖 **5大章节**剧情
- 😤 **6种情绪**: 喜、怒、哀、惧、爱、恶
- 🍚 **5种欲望**: 食欲、财欲、名欲、睡欲、玩欲
- 🏆 **30+成就**系统
- 🎲 **随机事件**系统
- 💾 **存档/读档**功能

## 📖 章节剧情

1. **初入社会** (0-30天) - 刚刚毕业，进入社会闯荡
2. **职场新人** (30-60天) - 努力工作，升职加薪
3. **创业之路** (60-100天) - 辞职创业，追求梦想
4. **人生巅峰** (100-200天) - 功成名就，颐养天年
5. **传奇人生** (200-365天) - 终极挑战，完美人生

## 🎯 通关条件

- **胜利**: 存活365天
- **失败**: 健康/精神/精力归零、情绪崩溃、没钱

## 🚀 如何运行

### 方法1: 直接运行HTML版本
直接用浏览器打开 `brain_game.html` 即可游玩

### 方法2: Cocos Creator项目
用 Cocos Creator 2.4.x 打开项目目录，构建发布

## 📁 文件说明

- `brain_game.html` - 完整版HTML游戏（直接可玩）
- `GameManager.js` - Cocos游戏主逻辑
- `GameCore.js` - 游戏核心数据配置

## 🛠️ 技术栈

- Cocos Creator 2.4.x
- JavaScript (ES6)
- HTML5/CSS3

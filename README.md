# 🧠 七情六欲大脑模拟器

人生365天精品版 - Cocos Creator项目

## 项目结构

```
brain-simulator/
├── assets/
│   ├── scenes/
│   │   ├── Main.fire      # 主游戏场景
│   │   ├── Menu.fire      # 菜单场景
│   │   ├── GameOver.fire # 游戏结束场景
│   │   └── Win.fire      # 通关场景
│   └── scripts/
│       ├── GameManager.js # 主游戏逻辑
│       └── GameCore.js    # 游戏核心数据
├── settings/
│   └── builder.json       # 构建配置
└── project.json           # 项目配置
```

## 游戏特色

### 🎮 核心玩法
- 5大章节剧情
- 7种天赋选择
- 6种情绪系统
- 5种欲望系统
- 30+成就系统
- 随机事件系统

### 📖 章节剧情
1. 初入社会 (0-30天)
2. 职场新人 (30-60天)
3. 创业之路 (60-100天)
4. 人生巅峰 (100-200天)
5. 传奇人生 (200-365天)

### 🏆 通关条件
- 胜利: 存活365天
- 失败: 健康/精神/精力归零、情绪崩溃、没钱

## 使用方法

1. 用Cocos Creator 2.4.x打开项目
2. 构建发布到Web Mobile
3. 或直接在浏览器中运行

## 直接可玩版本

桌面上有完整版HTML游戏可以直接玩:
- `七情六欲精品版.html`

## 技术栈

- Cocos Creator 2.4.x
- JavaScript (ES6)
